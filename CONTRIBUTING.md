# Contributing to Ebola Resources

Thank you for helping build this resource collection for the Ebola Community of Practice.

For full details, see the [Contributing guide](https://bafnaprincy.github.io/ebola-resources/contributing/) on the site.

## Quick start

- **Suggest a resource (no coding):** [Open an issue](https://github.com/bafnaprincy/ebola-resources/issues/new?template=new-resource.yml)
- **Add narrative context:** Edit a section page under "Community Notes" and open a PR
- **Add a structured entry:** Edit `data/resources.yml` and open a PR

## For maintainers

See the [Maintainers guide](https://bafnaprincy.github.io/ebola-resources/maintainers/) for the full workflow.
